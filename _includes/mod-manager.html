Simple Mod Manager

This script helps Arma Reforger server administrators check and format their server modlist in JSON format. The generated output can be copied to the clipboard for easy use.

Usage

  1. Upload Your Modlist File:
    • Prepare a text file containing your Arma Reforger server modlist, a line separated list of arma reforger workshop URLs. Use hash comments to exclude mods from configuration output (e.g., modlist.txt).
    • Click the "Check Modlist" button and select your modlist file.
  2. Review and Copy:
    • The script will process your modlist, and the formatted JSON output will be displayed in the result box.
    • Click the "Copy to Clipboard" button to copy the generated JSON to your clipboard.
  3. Paste into Server Configuration:
    • Paste the copied JSON into your Arma Reforger server mod configuration where the modlist is specified.
